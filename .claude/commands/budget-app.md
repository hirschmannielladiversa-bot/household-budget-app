# 家計管理アプリを起動

家計管理ダッシュボードを起動します。

## 実行内容

1. プロジェクトディレクトリに移動
2. Streamlitアプリを起動
3. ブラウザで http://localhost:8501 が開きます

## 起動コマンド

```bash
cd "/Volumes/tkg SSD/household-budget-app" && python3 -m streamlit run app.py
```

## 使い方

1. **データ読み込み**: サイドバーからCSV/Excelをアップロード、または「サンプルデータを読み込む」
2. **グラフ確認**: 「📈 グラフ」タブで支出を可視化
3. **AIアドバイス**:
   - サイドバーでAPIキーを入力
   - 「Claudeによる自然文アドバイスを有効化」にチェック
   - 「🧭 アドバイス」タブでチャット相談

## 停止方法

ターミナルで `Ctrl + C`
