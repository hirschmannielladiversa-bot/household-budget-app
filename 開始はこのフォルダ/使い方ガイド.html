1. アプリを起動する

初回はターミナルから起動してください
macOS では、ダウンロードした .command ファイルをダブルクリックや右クリック→「開く」で実行しようとすると、 Gatekeeper（セキュリティ機能）によりブロックされて正常に開けないことがあります。 特に macOS Ventura / Sequoia 以降で制限が厳しくなっています。
一度ターミナルから起動すれば、2回目以降はダブルクリックでもOKになります。
事前準備（初回のみ）：Python 3.12 のインストール
python.org から Python 3.12 をダウンロード・インストールしてください。 最新の 3.13 / 3.14 は一部パッケージが未対応の可能性があります。

初回：ターミナルから起動する（推奨）

1ターミナルを開く
Finder →「アプリケーション」→「ユーティリティ」→ ターミナル.app
または Spotlight（Cmd + Space）で「ターミナル」と入力
2アプリのフォルダに移動する
ターミナルに以下を入力します（フォルダの場所に合わせて変えてください）：
cd ~/Desktop/household-budget-app-v2.5.1-Mac
ヒント: cd と打ったあとに、Finder からフォルダをターミナルにドラッグ＆ドロップするとパスが自動入力されます
3必要なパッケージをインストール（初回のみ）
pip3 install -r requirements.txt
エラー externally-managed-environment が出た場合：
pip3 install --break-system-packages -r requirements.txt
4アプリを起動する
python3 -m streamlit run app.py
ブラウザが自動で開き、家計ダッシュボードが表示されれば成功です。
開かない場合は http://localhost:8501 にアクセスしてください。
配布版を初めて使う時のおすすめ手順
起動したら → 「サンプル（山田太郎）」を選んで機能を試す → 慣れたら「ユーザー切り替え」→「自分のデータ」で自分の家計を入力

2回目以降：ダブルクリックで起動

一度ターミナルから起動に成功すれば、次回からはこのフォルダにある 「家計アプリ起動.command」 をダブルクリックするだけで起動できます。

1 「家計アプリ起動.command」をダブルクリック
2 黒い画面（ターミナル）が出て、自動でブラウザが開きます
3 ブラウザに家計ダッシュボードが表示されたら準備完了！
「開発元が未確認のため開けません」と出たら：
→ ファイルを右クリック →「開く」→「開く」をクリックしてください（初回のみ）。

終了するには

ターミナルで Ctrl + C を押すか、ターミナルウィンドウを閉じてください。

4. 収入を入力する

手動で入力する場合

1「💰 収入管理」タブを開く
2 月ごとの「収入（円）」欄に金額を入力
3「💾 収入データを保存」をクリック

口座から自動で取得する場合

1 まず「💳 口座管理」で銀行CSVを取り込む
2「💰 収入管理」タブを開く
3「🔗 口座から給与データを検出」が表示される
4「📥 検出した給与データを収入に反映」をクリック
給与として検出されるのは、金額がプラスで摘要に「給与」と書かれた取引です。

5. 口座管理を使う

口座を追加する

1「💳 口座管理」→「➕ 口座追加」タブ
2 口座の種類（銀行/証券/カード）を選ぶ
3 口座名・銀行名・初期残高を入力
4「追加」ボタンを押す

CSV取り込み

1「📥 CSV取り込み」タブ
2 取り込み先の口座を選ぶ
3 銀行のCSVフォーマットを選ぶ（三菱UFJ、楽天銀行など）
4 CSVファイルをアップロード

PDF・画像取り込み（Gemini API必要）

1「📄 PDF・画像取り込み」タブ
2 取り込み先の口座を選ぶ
3「📄 PDF取り込み」または「🖼️ 画像取り込み」を選ぶ
4 ファイルをアップロード（複数可）
5「🔍 解析してインポート」をクリック
PDF・画像取り込みにはGemini APIキーが必要です。サイドバーで設定してください。

8. セキュリティについて

このアプリはあなたの大切なお金の情報を扱います。以下の仕組みで安全に守っています。

保護項目対策内容
データの保存場所すべてあなたのパソコン内のみ。クラウドには送信しません。
AI使用時のデータ金額は概算に丸め、口座名や保険会社名は送信しません。
APIキー画面に表示されず、ブラウザを閉じると消えます。
エラー表示内部の技術情報は表示せず、わかりやすいメッセージのみ表示します。
Git管理個人の金融データは.gitignoreで除外されています。
資産データパスワードで暗号化して保存できます。
AI機能を使わなければ、すべての処理がパソコン内で完結します。インターネット不要です。

9. よくある質問

Q. データが表示されません

→ サイドバーでCSVファイルをアップロードするか、「サンプルデータを読み込む」ボタンを押してみてください。

Q. カテゴリを変更したい

→「📄 データ一覧」タブで、カテゴリ欄をクリックするとドロップダウンで変更できます。変更後「💾 変更を保存」を押してください。

Q. データを間違えて入力した

→「📄 データ一覧」タブで該当行の「🗑️」にチェック → 「選択削除」ボタンで削除できます。

Q. 画面の表示がおかしい

→ ブラウザで Ctrl + R（Mac: Cmd + R）でページをリロードしてください。

Q. 口座のカテゴリが間違っている

→「💳 口座管理」→「📋 取引履歴」→「🔄 カテゴリ再分類」ボタンを押すと、最新のルールで全件再分類されます。

Q. PDF取り込みで「APIキーの利用制限」と出る

→ Gemini APIの無料枠の上限に達しています。少し時間を空けて再試行するか、有料プランに切り替えてください。

10. アプリのカスタマイズ（Claude Code）

このアプリは Claude Code（AIプログラミングアシスタント）を使って、 かんたんにカスタマイズできます。プログラミングの知識は必要ありません。 日本語でやりたいことを伝えるだけで、Claude Codeがコードを編集してくれます。

カテゴリ分類がうまくいかない場合

たとえば「スターバックス」が食費ではなく娯楽費に分類されてしまう場合、 Claude Codeに以下のようにお願いするだけでOKです。

あなた：
「スターバックス」という字を含むものは食費にカテゴライズしてください

あなた：
「ENEOS」を含む取引を交通費に分類してください

あなた：
「Netflix」と「Disney」は娯楽費にしてください
Claude Codeは config/bank_formats.yaml というファイルを編集して、カテゴリ分類のルールを自動で更新してくれます。

使いにくい部分を改善したい場合

「ここが使いにくい」「こういう機能がほしい」と思ったら、 そのまま Claude Code に伝えてください。

例：
・グラフの色を変えてほしい
・新しいカテゴリ「ペット費」を追加してほしい
・月別の推移グラフに平均線を入れてほしい
・データ一覧を金額の大きい順に並べたい
・口座管理に〇〇銀行のCSVフォーマットを追加してほしい
Claude Codeは、このアプリのフォルダで起動してください。 ターミナルで cd household-budget-app のあと claude と入力すると Claude Code が起動します。

大事な注意：セキュリティチェック

Claude Codeに編集をお願いした後は、 必ずセキュリティチェックも一緒にお願いしてください。

コードを変更すると、意図せずセキュリティに穴ができることがあります。 編集が終わったら、同じ会話の中で以下のようにお願いしましょう。

あなた：
今の変更でセキュリティ上の問題がないか確認してください。
APIキーの漏洩リスクや、個人情報の保護も含めてチェックをお願いします。

Claude Code は変更した箇所を自動で検出し、以下のようなチェックをしてくれます。

チェック項目内容
APIキーの漏洩APIキーがコードに直接書かれていないか
個人データの保護名前や口座番号がGitに含まれないか
エラーメッセージ内部情報が画面に表示されないか
入力の安全性悪意のあるデータが入力されても壊れないか
AI送信データ外部APIに送る情報が最小限になっているか
まとめると、Claude Codeへのお願いは2ステップです：
① やりたいことを伝える（例：「〇〇を食費にして」）
② セキュリティチェックをお願いする（例：「セキュリティも確認して」）

11. APIキーって何？（くわしい説明）

そもそもAPIキーとは？

APIキーは、外部のAIサービスを使うための「合鍵」のようなものです。

たとえば、Netflixを見るにはアカウントとパスワードが必要ですよね。 同じように、AIの機能を使うには「APIキー」という特別な文字列が必要です。

APIキーは英数字の長い文字列で、たとえばこんな見た目です：
sk-ant-api03-xxxxx...xxxxx（実際はもっと長い）

APIキーがなくても、このアプリの基本機能（データ入力、グラフ、カテゴリ分類、収入管理、ローン管理など）はすべて使えます。APIキーが必要なのはAI機能だけです。

このアプリで使えるAPIキーは3種類

APIキー提供元何に使う？取得場所
Claude API Anthropic社 家計アドバイス、チャット相談 console.anthropic.com
Gemini API Google レシート読み取り、PDF/画像取り込み、アドバイス aistudio.google.com
Google Sheets API Google Googleスプレッドシートとの連携 console.cloud.google.com

無料？有料？料金のしくみ

🟢 Gemini API（おすすめ：無料枠あり）

プラン料金制限
無料枠0円1日あたりのリクエスト数に上限あり（約15回/分、1500回/日）
有料（従量課金）使った分だけ上限が大幅に緩和。月数百円〜数千円程度

レシート読み取りやPDF取り込みに使います。無料枠でも十分試せますが、たくさんのPDFを一度に取り込むと上限に達することがあります。その場合は少し時間を空けてください。

🔵 Claude API（高品質なアドバイス）

プラン料金特徴
無料枠初回 $5分のクレジット付与アカウント作成時にもらえる（キャンペーンにより変動）
従量課金使った分だけ1回のアドバイス生成で約 ¥5〜¥30 程度

家計の総合アドバイスやチャット相談に使います。質の高い日本語アドバイスが得られますが、利用には課金が必要になる場合があります。

🟠 Google Sheets API（無料）

プラン料金特徴
無料0円Googleアカウントがあれば無料で使える

Googleスプレッドシートから家計データを読み込むのに使います。設定がやや複雑（サービスアカウントの作成が必要）ですが、完全に無料です。

APIキーの取得手順

Gemini APIキー（一番かんたん）

1 Googleアカウントでログイン
2 Google AI Studio（aistudio.google.com）にアクセス
3「Get API key」→「Create API key」をクリック
4 表示されたキーをコピー
5 アプリのサイドバーにある「Gemini APIキー」欄に貼り付け

Claude APIキー

1 console.anthropic.com でアカウント作成
2 クレジットカードまたはデビットカードを登録（従量課金のため）
3「API Keys」→「Create Key」をクリック
4 表示されたキーをコピー（sk-ant-... で始まる文字列）
5 アプリのサイドバーにある「Claude API Key」欄に貼り付け

APIキーの安全な管理

APIキーは絶対に他人に教えないでください。 キーが漏れると、他の人があなたのアカウントでAIを使い、あなたに料金が請求されます。
やっていいことやってはいけないこと
アプリのパスワード欄に貼り付けるSNSやメールに書く
使わないときはAPIキーを消すGitHubなどに公開する
環境変数に設定する（上級者向け）他人のパソコンに入力したまま放置
環境変数で設定する方法（上級者向け）：
ターミナルの設定ファイル（~/.zshrc など）に以下を追加すると、毎回入力しなくて済みます：
export ANTHROPIC_API_KEY="sk-ant-api03-xxxxx"
設定後はターミナルを再起動してからアプリを起動してください。
